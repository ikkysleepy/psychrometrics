Psychrometrics allows the calculation of psychrometric properties of moist and dry air.

Here are the available functions:

  1. dewpoint - Calculates dew point based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be wet-bulb (Situation 1), relative humidity (situation = 3), or humidity ratio (Situation=4).

  2. enthalpy - Calculates enthalpy based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be wet-bulb (Situation 1), dew-point (situation = 2), relative humidity (situation = 3), or humidity ratio (Situation=4).

  3. humidity - Calculates relative humidity (%) based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be wet-bulb (Situation 1), dew-point (situation = 2), or humidity ratio (Situation=4).

  4. humidratio - Calculates humidity ratio based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be wet-bulb (Situation 1), dew-point (situation = 2), or relative humidity (situation = 3).

  5. volume - Calculates volume based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be wet-bulb (Situation 1), dew-point (situation = 2), relative humidity (situation = 3), or humidity ratio (Situation=4).

  6. wetbulb - Calculates wet bulb based on T (°F or °C), P (psia or kPa) and state vairable. The state variable can be dew-point (situation = 2), relative humidity (situation = 3), or humidity ratio (Situation=4).

  7. altitudeAtmoP - Calculates estimate of atmospheric pressure from altitude (ft or km).

Example: Finding Relative Humidity

Given:

◦ Dry-bulb temperature 20 °C

◦ Wet-bulb temperature 12 °C

◦ Pressure 101.325 kPa

◦ Situation 1: (wet bulb is knonwn)

◦ Unit: SI

Find Relative Humidity using Psychrometrics functions:

◦ humidity( dew-point, pressure, situation, wet-bulb, si_units)

   Formula =psychrometrics.humidity( 20, 101.325,1, 12, TRUE)
